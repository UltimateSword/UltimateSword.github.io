# 基础数据结构及实现
## 线性表
1. Linked List（链表）：插入，删除速度快O(1)。查找速度慢O(n)

Python
```Python
class ListNode:
    def __init__(self, val):
        self.val = val
        self.next = None
```
C++
```C++
struct ListNode {
    int val;
    ListNode *next;
    ListNode(int val,ListNode *next=NULL):val(val),next(next){}
};
```
链表翻转
```Python
def reverse(head):
    res = None
    while head:
        tmp = head.next
        # 解锁
        head.next = res
        # 赋值 循环不变式
        res = head
        head = tmp
```
